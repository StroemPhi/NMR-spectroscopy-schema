Class: NmrSpecRecordCollection

This class represents a collection of NMR spectroscopy records. It is to be used when multiple NMR assays were made in order to analyse a given sample.

URI: NMRspec:NmrSpecRecordCollection

Uses Mixin

  • mixin: NamedThing - a mixin to be used by any class that has a name and unique id
